## Kort beskrivelse

Brukeradministratorer i Sikt og ved lærestedene kan se relevante maskinbrukere og kontaktpersoner for å ivareta sikkerhet og personvern.

- Innlogget bruker med `brukeradmin_les_alle_larested` ser alle maskinbrukere og kontaktpersoner.
- Innlogget bruker fra lærested med `brukeradmin_les_eget_larested` ser maskinbrukere og kontaktpersoner for eget lærested.
